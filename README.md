**// sobre**

<img align="right" src="https://media.giphy.com/media/836HiJc7pgzy8iNXCn/giphy.gif" width="180" />

Engenheiro de Dados especializado em **arquitetura Lakehouse, PySpark e Databricks** aplicados ao setor financeiro. Construo pipelines que movem dados de onde estão para onde precisam estar — limpos, rápidos e confiáveis.

5 anos de experiência no setor financeiro com histórico de reduzir lead time de processamento em **95%** (de 6h para 10min) via migração de sistemas legados para arquiteturas modernas em cloud.

> *"Não construo pipelines por construir — cada solução começa com um problema real de negócio."*
